# DesafioZG - Player de Música Interativo

## Descrição

Este projeto é um player de música interativo que permite aos usuários ouvir a música "Erguei As Mãos" em partes distintas, visualizar a letra da música, e interagir com a interface de usuário.

## Funcionalidades

- Tocar a música inteira
- Tocar somente a primeira parte da música
- Tocar somente a segunda parte da música
- Visualizar a letra da música no Console
- Barra de progresso para acompanhar o tempo da música

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
